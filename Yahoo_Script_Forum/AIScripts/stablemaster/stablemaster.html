Stable Master - version .18
Updated by RoTTer Dec 8

This is RoTTer's Stable Keeper aka Puppies Collector script. You can place the Stable Masters around the country and players can stable their pets by using the command stable. They are given a ticket they need to give back to claim their pet. Players can claim their pet from any stable master, not just the one they took their pet to.

How to set up the script:

  1. Copy the stamaster.src and stamaster.ecl to the \pol\scripts\ai directory.
  2. GM's can use the command .createnpc stablemaster
  3. Add the following item to the file \pol\config\npcdesc.cfg
NpcTemplate stablemaster
{
Name the Tamer
script stamaster

ObjType 0x190
Color 33784
TrueColor 33784
Gender 0
STR 200
INT 200
DEX 200
HITS 200
MANA 200
STAM 200

Tactics 100
MaceFighting 140

AttackSpeed 55
AttackDamage 2d100
AttackSkillId MaceFighting
AR 32
Privs invul
Settings invul

MagicResistance 120
guardignore 1
dstart 10
}
