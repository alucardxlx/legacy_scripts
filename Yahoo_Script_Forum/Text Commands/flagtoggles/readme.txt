Flag Toggles v1.0
HellRazor (hellrazor@hellrazor.net)

Requires POL v91 or higher.

.murdererflag
.criminalflag

These commands will toggle criminal flags and murderer flags.  If they are already flagged, the command will turn the flag off.

Installation:

Unarchive these commands into your scripts/textcmd/gm directory (or whatever command level you wish to allow to use the commands), then compile with ecompile.
