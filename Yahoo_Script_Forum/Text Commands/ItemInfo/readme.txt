======================================================================
Contents, by Version:
*** v1.1 ***

	Added Item Quality. Fixed some problem when duping Items.

*** v1.0 ***

	Possibility to see and modify values of Items proprerty
	
	Base commands are : x - y - z coords , Item Color , Item 
	Graphic , Dupe item , create a stack in backpack of selected
	Item, change Name , HP/MAXHP , Bounce Object , Destroy Item.

	Advanced Options : Set Item Newbie or not , Set Item Movable or
 	not. Support to Change AR value for Armors ( ar_mod ) and Change 
	Damage for ( dmg_mod ) directly in game.

	Note :POL doesn't allow to edit Items equipped by other players.
