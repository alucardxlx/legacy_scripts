Random Tips v1.0
HellRazor (hellrazor@hellrazor.net)

This package will display a random tip to your users when they log on.

Installation:
-------------
1.  Unarchive the zipfile into your PKG/OPT directory.

2.  Compile the scripts in PKG/OPT/RANDOMTIPS and PKG/OPT/RANDOMTIPS/TEXTCMD/PLAYER using Ecompile.

3.  You can customize the tips by opening PKG/OPT/RANDOMTIPS/TEXTCMD/PLAYER/TIPS.CFG.  Enter the total number of tips in the maxtips section, then enter the text for each tip in the same format as the examples already included.
